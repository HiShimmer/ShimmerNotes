# markdown学习笔记
使用的工具有Markdown Monster和typora,不仅是学习markdown，而且是一个编辑器的学习。

Markdown Monster中
* To create a **New Document** press **Ctrl-N** or **click @icon-plus-circle** on the toolbar

* To toggle the **HTML Preview** press **F12** or click @icon-globe in the Window bar

## 标题
markdown中的title要在纯文本之前添加 “#” ，“#” 的数量是标题的级数。最大六级，一级标题二级标题中有分割线。

# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6

### 加粗 and 斜体 and 删除线
This text **is bold**.  
This text *is italic*.  
This text ~~is struck out~~.
### 缩小字体
尖括号中small，类似html中的格式。
<small>(shift-enter)</small> 
or use a backslash `\`.
`这是什么符号？，哦，是TAB上面那个`

这是👇分割线
